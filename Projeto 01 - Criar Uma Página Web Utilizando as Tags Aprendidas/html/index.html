O que é HTML?

Voltar

Antes de começarmos a conhecer algumas tags básicas do HTML, vamos entender o que é essa linguagem. HTML é uma linguagem de marcação de hipertexto.É ela que define como os elementos serão estruturados em website, uma comparação ideal seria como se fosse o esqueleto do site, uma das definições do órgão W3C é

Os elementos HTML rotulam partes de conteúdo como "este é um título", "este é um parágrafo", "este é um link", etc.

Algo que é muito importante e vale a pena destacar é que a linguagem HTML não trata-se de uma linguagem de programação, não possuindo lógica, apenas realizando a marcação do conteúdo

O HTML foi criado por Tim Berners-Lee no final dos anos 1980 e início dos anos 1990. Ele é um cientista da computação britânico que trabalhava no CERN, na Suíça. Em 1991, a primeira página da web foi publicada e explicava o que era a World Wide Web

Tags de listas

Voltar

Se tratando de tags de listas temos 2 tags para essas situações as ul e as ol

As listas ordenadas são geralmente utilizadas quando a ordem é importante por exemplo uma receita:

  1. Esquentar o leite
  2. Colocar creme de leite
  3. Colocar chocolate
  4. Mexer até ficar consistente

As listas não ordenadas são geralmente utilizadas quando a ordem não é importante por exemplo uma lista de compras:

  • Café
  • Leite
  • Chocolate
  • Biscoito
